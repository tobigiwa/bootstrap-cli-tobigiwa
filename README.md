# 🚀 Bootstrap CLI

Bootstrapping your projects by one CLI.

![demo](https://user-images.githubusercontent.com/35857179/219651301-cfa1d385-b215-43c8-be65-68b88c1139d0.gif)

## 🛠 Prerequisites

- Go 1.19

## 💻 Quick Start

```go
go install github.com/wingkwong/bootstrap-cli@latest
```
